# Book Management

   Maintain book details in a library. Allows you to add/remove/update/get any book in the library

## Features
* Add a book
* Search a book
* Update attributes a book
* Remove a book


## Packages Used
* Express
* Mongoose
* SuperTest
* Swagger-ui-express
* Istanbul
* Http-status-code
* Sinon
* Should

## API documentation
API documentation could be reached at http://localhost:3333/api-docs/
